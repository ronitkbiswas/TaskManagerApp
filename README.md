# 📝 Task Manager App (React)

A simple and clean Task manager application built using React. This app allows users to add, delete, and manage daily tasks efficiently.

---
## Live Link
https://taskmanager-ronitkbiswas.vercel.app/

## 🚀 Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Clean and responsive UI
* Real-time updates using React state

---

## 🛠️ Tech Stack

* React (Functional Components)
* JavaScript (ES6+)
* CSS

---

## 🧠 Concepts Used

* useState (state management)
* Event handling
* Conditional rendering
* Component-based architecture

---

## 📂 Project Structure

src/
├── App.js
├── components/
│   ├── TaskInput.js
│   ├── TaskList.js
│   └── TaskItem.js


## 🎯 Purpose

This project was built to understand the fundamentals of React such as state management, component structure, and handling user interactions.
